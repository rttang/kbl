# kbl.js
##简介
keyboard library是一个为vod开发前端部分的一个键盘控制焦点的库，
想法很简单
- 每个焦点都有对应的html，每个html都有其id，然后以该id作为在js中的一个标志
- 这个标志（变量）中存储着上下左右以及可选的data，同时使用发布/订阅模式绑定自定义事件（focus、blur时的动作）
- 自动生成矩形的按钮集
- 本库结构写法和部分功能来源于jquery。
